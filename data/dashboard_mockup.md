# 운영 대시보드 시각화 계획

## 1. 상단 KPI 바
- SLA 준수율(티어별)
- 평균 처리시간(분)
- 저가 견적 경고 발생률
- 환불/보상 처리 건수

## 2. SLA 타임라인 영역
- 라인 차트: 티어별 처리시간 추이(24h, 7d)
- 히트맵: 지연 발생 시간대/요일

## 3. 워커 로드/QA 모듈
- 전문가별 진행 중 건수, SLA 잔여시간 표시
- QA 재검토 비율, 수정 요청 상위 항목

## 4. 고객 퍼널 & 채널 성과
- Funnel 차트: 방문 → 업로드 → 결제 → 리포트 열람 → 재의뢰
- 채널별 전환율/ARPU 테이블(필터)

## 5. 경고 패널
- 저가/고가 감지 목록(공사비, 퍼센트, 이유)
- 건설업 면허/보증보험 확인 필요 목록
- SLA 위반 티켓 리스트(우선순위)

## 6. 시스템/데이터 상태
- ETL 성공률, 마지막 갱신 시각
- 알림 발송 실패율, 재시도 상태

(대시보드 레이아웃은 2×3 그리드 배치를 기본으로 하며, KPI 바는 상단 고정)
