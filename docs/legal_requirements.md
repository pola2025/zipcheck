# 법무 · 보험 요구사항 세부안

## 1. 법무 체크리스트
- 약관/개인정보 처리방침 연 1회 이상 검토, 개정 시 7일 전 공지
- 환불 불가 정책과 SLA 보상 규정 법적 검토
- 디지털 분석 서비스 관련 전자상거래법/전자문서법 준수

## 2. 면허/보험 검증
- 공사비 1억 원 이상 프로젝트: 건설업 면허 사본, 보증보험 증명서 업로드
- 전문가 콘솔에 증빙 업로드 필드, 운영팀 검토 후 승인
- 미제출 시 경고 및 리포트 상 주의 문구 표시

## 3. 계약 문서
- 전문가 용역 계약: 기밀 유지, 책임 한계, 데이터 재사용 금지
- 제휴 계약: 데이터 공유 범위, 고객 동의, SLA 책임 분담
- 고객사(기업) 대상 B2B 계약: Service Level, 보상 규정, 책임 한계 명시

## 4. 기록 보관
- 계약서/증빙 자료는 전자문서 형태로 5년 보관, 접근 권한 제한
- 감사 로그 및 변경 이력은 `audit_logs`와 별도 보관소에 저장

## 5. 이의 제기 · 분쟁 처리
- 고객 이의 제기 절차: 접수 → 재검토 → 결과 통보 (72시간 내)
- 분쟁 발생 시 법무팀/경영진 협의, 필요 시 외부 법률 자문
- 분쟁 사례 및 해결 내역을 분기별 보고서로 작성

## 6. 보험 가입
- 회사: 영업 배상 책임보험, 사이버 보험 검토
- 전문가: 필요 시 개인 전문 indemnity 보험 안내
