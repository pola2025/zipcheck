# 마케팅 실행 계획

## 1. 목표
- 초기 6개월 내 유료 고객 500명 확보, 제휴 채널 5곳 연동
- CAC ≤ 150, LTV/CAC ≥ 3, 재의뢰율 30% 달성

## 2. 채널 전략
- **콘텐츠 마케팅**: 블로그(주 1회), 데이터 리포트(월 1회), 유튜브 숏폼(월 2회)
- **제휴 마케팅**: 이사/부동산 플랫폼과 공동 프로모션, 신규 가입 쿠폰 제공
- **유료 광고**: 검색/리타겟팅 캠페인, 저가 견적 문제 해결 메시지 강조
- **CRM/리텐션**: 리포트 발송 후 3·14·30일 follow-up, 재의뢰 쿠폰, 체험 후기 수집

## 3. 캠페인 로드맵
| 분기 | 캠페인 | 주요 콘텐츠 | KPI |
| --- | --- | --- | --- |
| 2026 Q1 | 런칭 캠페인 | 서비스 소개 랜딩, 사례 리포트 | 가입 수, 전환율 |
| 2026 Q2 | 저가 견적 위험 알림 | 데이터 리포트, 웨비나 | 리포트 다운로드, 상담 요청 |
| 2026 Q3 | 전문가 인터뷰 시리즈 | 전문가 노하우, 협상 가이드 | 콘텐츠 조회, 재의뢰율 |

## 4. 예산/측정
- 월간 예산 1,000만 원, 채널별 40%(콘텐츠) / 30%(제휴) / 20%(광고) / 10%(CRM)
- 측정: GA4, Looker Studio, `operational_metrics` 연동
- KPI 리뷰: 주간 스탠드업, 월간 리포트 공유

## 5. 프로세스
1. 콘텐츠 기획 → 제작 → 검수 → 배포 → 성과 분석 → 인사이트 공유
2. 제휴 채널: 계약 → 랜딩 제작 → 성과 트래킹 → 리포트 공유
3. 광고: 캠페인 셋업 → 2주 단위 최적화 → 리포트

## 6. 조직 및 역할
- 마케팅 리드: 전략, 예산 관리, KPI 보고
- 콘텐츠 매니저: 자료 제작, 블로그·영상 관리
- 퍼포먼스 매니저: 광고 집행, 분석
- 파트너 매니저: 제휴 협상, KPI 공유
